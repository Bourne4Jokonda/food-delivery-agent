# Food Delivery Agent

Простой агент для поиска по базе знаний доставки еды.

## Что умеет
- Извлекает теги вида `#слово` из запроса.
- Ищет релевантные документы сначала по тегам, затем через fuzzy-поиск (`fuse.js`).
- Формирует ответ через заглушку GigaChat.

## Запуск
```bash
cd food-delivery-agent
npm install
npm start
```

## Примеры запросов
- `Какие есть #пицца и #веган позиции?`
- `Как работает #доставка ночью?`
- `Можно ли оплатить картой?`
